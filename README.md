# Supply Chain Risk & Performance Analysis (SQL)

## 📊 Project Overview

This project analyzes logistics data to understand how environmental risks and driver behavior impact shipping costs, delays, and fuel consumption.

##  Tools Used

* SQL Server (SSMS)

##  Dataset

* Logistics & vehicle performance dataset

##  Key Skills

* Joins
* CTEs
* Window Functions
* Stored Procedures
* Data Cleaning

##  Key Insights

* High-risk conditions increase delays and costs
* Poor driver behavior leads to higher fuel consumption
* Monthly trends show fluctuations in logistics performance

##  Features

* Risk impact stored procedure
* Driver performance classification
* Data cleaning and validation queries
* Time-based analysis
